选择器

 同时多个使用多个选择器 ：
  使用 “，”分隔
 派生选择器 ：
  子关系；
  单重（严格）子关系（>）；
  兄弟关系(+);
 id
  # 作为标识符
  通常作为派生选择器　#itname p {}
 class
  .　作为标识符(.itname)
  可以用做派生选择

属性选择器

 title
  [title]
  [title = itname]
  [title ~= partname]
  在没有任何值的时候，对于所有含title的句子都渲染

Working with Text

font

 font-family:
  type of font
 font-size:
  1em == 16px
  notice: try different combinations of font-size and page zooming
 font-style:
  italic（寻找是否该字体存在斜体）
  normal (such as using in “strong”)
  oblique (强制渲染成斜体)
 font-weight:
  bold
  bolder(感觉和bold没有区别)
  lighter
  number from 100 to 900
 font-variant:
  normal
  small-caps(奇妙的变化：首先让原来的大写字母突出放大，再让小写字母变成大写)
  inherit(继承父元素的值)

color

 rgb（不会用）
 #0F(不会用×２)
 使用名字

text-align:

 left
 right
 center
 justify(实现两端对齐文本)

vertical-align

 table-tr-td:
  top
  middle
  bottom
 in one line
  baseline
  sub
  super
  10px or -10px(px , pt , cm)
 div
  height
  width
  background-color
  inline-table
  div:
   table-cell
   vertical-align:middle or else

text-decoration

 none
 inherit
 overline
 underline
 line-through

text-indent

 60px also -60px
 inherit

text-shadow

 Xpx Ypx blur(>=0) color(or default same color)

text-transform

 capitalize (首字母大写)
 uppercase
 lowercase

letter-spacing

 normal
 length (can also be negative)
 inherit

word-spacing

 normal
 length
 inherit

white-space

 normal （不处理换行符，必要时换行，空格不缩减）
 inherit
 nowrap (禁止换行，缩减空格)
 pre-wrap (不处理换行符，必要时换行，缩减所有空格)
 pre-line （处理换行符，缩减空格）
 pre （处理换行符，空格不缩减）

table

 property
  height
  width
  bolder: 1px solid blue;
 tr 行
 th , td 列
  th 自动加粗，td普通
